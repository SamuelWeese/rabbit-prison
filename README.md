# Rabbit Prison

A game where you play as the warden managing rabbit prisoners, inspired by The Escapists.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the game:
```bash
python3 main.py
```

## Controls

- **WASD** or **Arrow Keys**: Move the camera/view around the prison
- The warden (blue character with hat) and rabbits (brown characters with ears) are visible in the prison cells

## Features

- 2D movable game area with camera controls
- Prison walls and cell structures
- Warden character (blue)
- Rabbit prisoner characters (brown with ears)
- Grid overlay for reference
